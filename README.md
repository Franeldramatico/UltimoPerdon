# Para mi niña hermosa 🖤🎀

Una página web especial hecha con mucho amor y un toque gótico lindo para pedir perdón y recordarte lo mucho que te amo.

## 🛠️ Tecnologías
- **React 19** + **Vite**
- **Tailwind CSS v4** (Gothic-cute styling)
- **Lucide React** (Icons)
- **TypeScript**

## ✨ Características
- Animaciones fluidas y efectos de partículas de emojis.
- Diseño interactivo tipo "carta" con múltiples pasos.
- Estilo "Cute Goth" personalizado con tipografías de Google Fonts.

## 🚀 Cómo ejecutar
1. Instala las dependencias:
   ```bash
   npm install
   ```
2. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
3. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
